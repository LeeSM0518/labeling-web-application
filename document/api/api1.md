# 1차 API 설계

## 종이 크기 및 라벨 스타일 선택 페이지

* **URL** : /init
  * 종이 사이즈
  * 라벨 스타일
* **POST**
  * 종이 사이즈, 라벨 스타일을 session에 저장

<br>

## 라벨지 꾸미는 페이지

* **URL** : /decorate
  * 라벨 배경색
  * 가운데 Bar 두께
  * 가운데 Bar 색
  * 라벨 테두리 두께
  * 라벨 테두리 색
  * 폰트
  * 머리글, 중간글, 꼬리글
  * 글씨 크기
  * 글씨 색깔
* **GET**
  * 종이 사이즈, 라벨 스타일을 session에서 제거
* **POST**
  * 라벨 배경색
  * 가운데 Bar 두께
  * 가운데 Bar 색
  * 라벨 테두리 두께
  * 라벨 테두리 색
  * 폰트
  * 머리글, 중간글, 꼬리글
  * 글씨 크기
  * 글씨 색깔
  * 위의 옵션들을 session에 저장

<br>
